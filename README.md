# Audrey Knafo — Home Page Directions (Preview)

Static design prototypes for an Audrey Knafo Shopify storefront. Three home-page directions, built with her own photography and a working FR/EN toggle.

**Open `index.html`** for the comparison landing page, or jump directly to a direction:

- `home-a-cinematic/` — Cinematic Hero (dark, moody)
- `home-b-editorial-split/` — Editorial Split (warm paper)
- `home-c-series-grid/` — Series-First Grid (neutral gallery)

> Preview only. Imagery is low-resolution web copy; display font (Fraunces) is a stand-in for the production face. Not the production site.
